## v0.6.2

### Features

- **LavaSR audio enhancement**: Opt-in neural speech enhancement post-processing with three independent controls:
  - `enhance` — Vocos-based neural bandwidth extension to 48 kHz (2 ONNX sessions, ~55 MB)
  - `denoise` — UL-UNAS denoiser at 16 kHz (1 ONNX session, ~1.7 MB)
  - `outputSampleRate` — arbitrary target sample rate with smart algorithm selection (neural upscaling + conventional resampling)
- **sampleRate in output callback**: `outputArray` now includes a `sampleRate` field; also reported in `runtimeStats` (JobEnded event)
- **Per-job enhance/denoise toggle**: `enhance`, `denoise`, and `outputSampleRate` can be set per-utterance via `run()` input, with lazy ONNX session loading on first use

### Performance

- Enhancer: ~22x realtime on Apple Silicon CPU (46ms per 1s of 48 kHz audio)
- Denoiser: ~48x realtime on Apple Silicon CPU (9ms per 1s of 16 kHz audio)
- All flags default to off — zero performance cost when not used
